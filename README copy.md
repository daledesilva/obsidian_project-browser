# Obsidian Handwrite
A plugin for [Obsidian](https://obsidian.md) that replaces your new tab window with a browsable view of your files and folders as cards.

!! ***This plugin is not yet released.***<br/>
Please also note that while the repository is public, the plugin it is not currently open source.
